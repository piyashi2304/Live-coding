What is Java?

Java is a object oriented programming langusge that was developed by James Gosling atSun Microsystem,which is now acquired by Oracle.In todays time Java is one of the most diverse language.Java programming language is a high lavel object oriented programming language designed to have a few implementation dependencies as possible

Key features of Java programming language:

Here are some important feature of Java language.

  1. Java programming language is designed to build object oriented applications.
  2. Java is multithreaded language with automatic memory management.
  3. In Java there is a feature called WORA i.e, write once run anywhere after compilation.
  4. Java code can be run any computing platform.
